# مساعد الذكاء الاصطناعي في أبشر

## فكرة المشروع
إضافة مساعد ذكاء اصطناعي داخل منصة أبشر يكون قادر على فهم طلبات المستخدم مباشرة سواء بالكتابة أو بالصوت، وتنفيذ الخدمات الحكومية بشكل تلقائي وسلس.

المستخدم يكتب أو يتكلم مثل: "أبغى أجدد جوازي" والمساعد يقوم بالآتي:
1. التأكد من أن المستخدم مسجل الدخول.
2. فتح الخدمة المطلوبة داخل أبشر.
3. طلب البيانات الضرورية فقط بطريقة مبسطة.
4. إرسال الطلب بالنيابة عن المستخدم.
5. عرض النتيجة النهائية مباشرة.

---

## دعم الصوت لكبار السن وذوي الهمم
- يتيح وضع صوتي كامل لكبار السن وذوي الإعاقة البصرية.  
- المستخدم يستطيع التحدث فقط، والمساعد ينفذ الإجراءات صوتيًا دون الحاجة لاستخدام الأزرار أو القراءة من الشاشة.  
- الهدف: جعل الخدمات الحكومية أكثر شمولية وسهولة للجميع.

---

## التقنيات المستخدمة
- Python  
- Google Colab  
- مكتبات: Pandas، NumPy، Matplotlib  
- واجهات API الخاصة بأبشر (أو محاكاة الإجراءات)

---

## طريقة استخدام المشروع
1. افتحي ملف الـ Notebook أو الرابط المرفق للموقع المحاكي لأبشر.  
2. ستظهر لك واجهة المحاكي كما أرفقنا.  
3. استخدمي **المساعد الذكي "أبشر وتم"**، سواء بالكتابة أو الصوت.  
4. اكتبي أو قولي الخدمة التي تريدين تنفيذها، مثل: "أبغى أجدد جوازي".  
5. المساعد يقوم تلقائيًا بتنفيذ الخدمة وإظهار النتيجة لك مباشرة، دون الحاجة لإدخال كل البيانات يدويًا.
